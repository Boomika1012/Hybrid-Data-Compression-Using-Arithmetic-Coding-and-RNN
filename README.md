# Hybrid Data Compression Using Arithmetic Coding and Recurrent Neural Networks (RNN)

## 📌 Project Overview

This project presents a **Hybrid Lossless Data Compression System** that combines the statistical efficiency of **Arithmetic Coding** with the predictive capabilities of **Recurrent Neural Networks (RNNs)**. The proposed approach improves compression performance by learning data patterns and generating accurate probability estimates before applying Arithmetic Coding.

The system is evaluated against the traditional **Shannon-Fano Coding** algorithm to compare compression ratio, execution time, and overall compression efficiency while ensuring complete lossless reconstruction of data.

---

## 🎯 Objectives

- Develop an intelligent hybrid lossless data compression system.
- Improve compression efficiency using Recurrent Neural Networks.
- Compare the proposed approach with Shannon-Fano Coding.
- Reduce storage requirements while preserving data integrity.
